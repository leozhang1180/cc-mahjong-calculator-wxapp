## 🚀 部署步骤

### 第一步：注册小程序账号
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击「立即注册」→ 选择「小程序」
3. 按提示完成注册，获取 **AppID**

### 第二步：安装开发工具
1. 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 安装并用微信扫码登录

### 第三步：创建项目
1. 打开微信开发者工具
2. 点击「+」创建项目
3. 填写：
   - **项目名称**：长春麻将算分器
   - **目录**：选择本 `miniprogram` 文件夹
   - **AppID**：填入你的小程序 AppID（或选择测试号）
   - **不使用云服务**
4. 点击「确定」

### 第四步：预览和调试
1. 项目打开后，左侧模拟器即可看到效果
2. 点击「预览」→ 用手机微信扫码在真机测试
3. 使用「真机调试」可在手机上调试

### 第五步：发布上线
1. 点击工具栏的「上传」按钮
2. 填写版本号和描述
3. 登录 [微信公众平台](https://mp.weixin.qq.com/) → 管理 → 版本管理
4. 提交审核 → 等待审核通过 → 发布

## 算分规则
- 1. 基本规则
4人对局，共136张牌
底分可调：0.5/1/2/5/10元
翻数制：胡分 = 底注 × 2^N（N为累加翻数）
可设封顶翻数（默认5番 = 32倍底注）
若累加翻数超过封顶，按封顶翻数计算
- 2. 胡牌方式
平胡：0翻（胡分为基础底注）
自摸：+1 番， 三家均付
摸宝：+2 番，三家均付
点炮：不加番，点炮者付包三家
- 1. 蛋的种类（任意满足一种即为1个蛋）：
常规杠牌：4张一样的牌（明杠/暗杠）。
三风蛋：东南西北风中，任意三张组合。
中发白：红中、发财、白板各一张组合。
幺九蛋：特定的幺九牌组合（如1万/9万等按当地约定组合）。
【特殊特权】幺鸡（1条）百搭：
幺鸡可以替代上述任何一种蛋里所需的任意一张牌。
三个幺鸡凑在一起，直接算作一个全能蛋。
2. 蛋分计算公式：
基础底分：任何形式的蛋（包含攒蛋后补下的蛋），基础分固定为 1分。
翻倍规则（可叠加相乘）：
暗蛋（未亮明，自己摸齐）：分数 × 2。
大蛋（满编大蛋，如凑齐4张风、4张中发白、4张幺九蛋等）：分数 × 2。
具体分值对照表：
普通明蛋（明杠/三风明蛋/攒出的明蛋）：1分
普通暗蛋（暗杠）：1 × 2 = 2分
大蛋（明）：1 × 2 = 2分
大蛋（暗）：1 × 2(暗) × 2(大蛋) = 4分
二、 胡分计算体系（牌面分）
胡分计算公式：胡分 = 底注 × (2 的 N次方) ，N为累加的翻数。
1. 基础牌型与加翻：
平胡：0翻（胡分为基础底注）。
通宝翻番（摸宝）：胡的牌刚好是翻开的宝牌，+1翻。（如单钓宝中宝，可做额外选项 +2翻）。
杠上开花：下蛋/开杠后补牌立刻胡牌，+1翻。
海底捞月：抓牌墙最后一张牌胡牌，+1翻。
2. 特殊大牌型（固定翻数）：
清一色：牌面全为同一花色（带不带字牌看房间设定），通常 +4翻。
七对：14张牌由7个对子组成，通常 +4翻。
豪华七对：七对中包含4张相同的牌，通常 +6翻 或 +8翻。
3. 满贯/封顶牌型（直接按设定的最高翻数计算）：
天胡：庄家起手14张牌直接胡牌。
地和：闲家听牌后，胡庄家打出的第一张牌。
(注：如果常规牌型加翻总和超过了房间设置的“封顶翻数”比如7翻，则也按封顶分数计算。)
三、 规则互斥情况（防错逻辑）
在小程序中，用户勾选时必须做以下互斥拦截，避免出现“假胡牌组合”：
天胡/地和 互斥 杠上开花/海底捞月（起手就胡，不可能存在补杠和摸底牌）。
海底捞月 互斥 杠上开花（长春麻将最后一张牌不允许下蛋/杠牌）。
七对/豪华七对 互斥 任何需要特定顺子/刻子的牌型（七对结构特殊）。

## 游戏规则

- **底分**: 每番基础分值（默认1分，实际得分 = 底分 × 2^番数）
- **包三家**: 点炮者替代其他两家付款
- **蛋分**: 独立于胡牌的积分，每局累计后一并结算
- **庄家规则**: 庄家赢或庄家点炮时分值翻倍；若庄家未胡牌则轮庄
